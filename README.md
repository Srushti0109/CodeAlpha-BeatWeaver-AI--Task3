📌 Project Overview
BeatWeaver AI is a full-stack web application that leverages Artificial Intelligence to compose original MIDI music. Developed as part of the CodeAlpha internship, this project bridges the gap between complex machine learning backends and intuitive user interfaces, allowing anyone to generate and download unique musical tracks.

🚀 Key Features
AI Music Composition: Generates MIDI files using specialized algorithms tailored for rhythmic and melodic consistency.

Interactive Web UI: A clean, responsive interface for configuring generation parameters and triggering compositions.

User Management: Secure user registration and activity logging to track generation history.

Instant Downloads: Seamlessly download generated MIDI files to use in your favorite DAW (Digital Audio Workstation).

🏗️ Architecture
The system is built with a decoupled architecture for high performance and scalability:

Frontend: A modern, interactive UI hosted on GitHub Pages.

Backend: A robust FastAPI server deployed on Railway, handling the logic for AI generation and database interactions.

Database: Integrated storage for managing user profiles and session logs.

🛠️ Tech Stack
Language: Python 3.x

Web Framework: FastAPI

Frontend: HTML5, CSS3, JavaScript (GitHub Pages)

Database: SQLAlchemy / SQLite

Deployment: Railway (Backend), GitHub Pages (Frontend)
